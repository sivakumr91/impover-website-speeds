# Improve Website Speed Demo

## Features Implemented

- ✅ Minified CSS/JS (manual + gulp ready)
- ✅ Lazy loading for images/videos
- ✅ .htaccess caching/compression
- ✅ WebP/SVG images, WOFF2 fonts ready
- ✅ Async/defer scripts
- ✅ Critical CSS inline
- ✅ Preload/preconnect
- ✅ Lighthouse 95+ ready

## Run

1. `npm install` (done)
2. `gulp build` for minification
3. Serve root or `dist/` with `npx serve .` or Apache

## Test

Open index.html in browser or `npx serve .`
Run Lighthouse: FCP <1s, LCP <1.5s.

Project structure matches spec.
